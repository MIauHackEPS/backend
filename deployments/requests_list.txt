Requests and example curl commands

AWS (terminal):
 - Create: bash aws_create.sh
 - List (GET): bash aws_list.sh
 - Find: bash aws_find.sh
 - Delete: bash aws_delete.sh

AWS (HTTP bodies for Astro):
 - POST /aws/create -> deployments/astro/aws_create.json
 - POST /aws/list   -> deployments/astro/aws_list.json (or GET /aws/list?region=us-west-2)
 - POST /aws/find   -> deployments/astro/aws_find.json
 - POST /aws/delete -> deployments/astro/aws_delete.json

GCP (terminal):
 - Create: bash gcp_create.sh
 - List: bash gcp_list.sh
 - Find: bash gcp_find.sh
 - Delete: bash gcp_delete.sh

GCP (HTTP bodies for Astro):
 - POST /create -> deployments/astro/gcp_create.json
 - POST /list   -> deployments/astro/gcp_list.json (or GET /list?credentials=...&zone=...)
 - POST /find   -> deployments/astro/gcp_find.json
 - POST /delete -> deployments/astro/gcp_delete.json

Notes:
 - AWS defaults: region `us-west-2`, AMI `ami-03c1f788292172a4e`, instances will be tagged with `Name` prefix `t3-`.
 - Deletion only removes instances with `Name` starting with `t3-`.
 - The API exposes GET endpoints to list instances for easier integration with Astro or frontend.
